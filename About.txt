Name:Ritchey Website Template
Implementation:4
Version:3
Long Identifier:Ritchey Website Template i4 v3
Short Identifier:RWTi4v3
Description:A basic website template written in HTML, Javascript, and CSS. It can be served from static-file only hosts, provides standard website pages (e.g., About Us, Terms of Service, Privacy Policy, etc), allows the addition of custom pages (referred to as posts), and has a built in basic search feature.
History:
 - Project started August 26, 2023.
 - Version 1 released September 1, 2023.
 - Version 2 released September 5, 2023.
 - Version 3 released September 12, 2023.
